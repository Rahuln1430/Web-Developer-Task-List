Flask Backend Overview

Flask is a lightweight Python framework used to build REST APIs and web applications.

Project Structure

flask-project/
│
├── app.py
├── requirements.txt
├── templates/
│   └── index.html

Example Flask Backend (app.py)


from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Welcome to Flask Backend"

@app.route("/api/users")
def users():
    return {
        "status":"success",
        "users":[
            {"id":1,"name":"Rahul"},
            {"id":2,"name":"John"}
        ]
    }

if __name__ == "__main__":
    app.run(debug=True)

Install Flask


pip install flask

Run Application


python app.py

Output

  • Home: http://127.0.0.1:5000/
  • API: http://127.0.0.1:5000/api/users

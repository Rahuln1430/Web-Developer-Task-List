Docker Setup Guide

Install, Configure & Run Docker Containers

Step 1 - Install Docker

Install Docker using your operating system package manager.

# Ubuntu sudo apt update sudo apt install docker.io

Step 2 - Start Docker

sudo systemctl start docker sudo systemctl enable docker

Step 3 - Verify Installation

docker --version docker run hello-world

Common Docker Commands

docker images docker ps docker ps -a docker pull nginx docker run -d -p 80:80 nginx docker stop container_id docker rm container_id

MongoDB Database Connection

HTML cannot directly connect to MongoDB. A backend server manages the database connection.

Example (Node.js + Mongoose)


const mongoose = require("mongoose");

mongoose.connect("mongodb://127.0.0.1:27017/mydatabase")
.then(() => {
    console.log("MongoDB Connected");
})
.catch(err => {
    console.log(err);
});

Connection Details

  • Database: mydatabase
  • Host: 127.0.0.1
  • Port: 27017
  • Driver: Mongoose

Application Architecture

Browser (HTML/CSS/JavaScript)
           │
           ▼
Backend (Node.js / Express)
           │
           ▼
        MongoDB

Workflow

  1. User submits a form from the HTML page.
  2. JavaScript sends an HTTP request to the backend.
  3. The backend validates the request.
  4. The backend reads or writes data in MongoDB.
  5. The backend returns a JSON response to the browser.

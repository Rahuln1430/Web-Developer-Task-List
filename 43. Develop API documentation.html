REST API Documentation

Version 1.0

Base URL

https://api.example.com/v1

Authentication

All protected endpoints require a Bearer Token.

Authorization: Bearer YOUR_ACCESS_TOKEN
GET

/users

Returns all users.

Response

[
  {
    "id":1,
    "name":"Rahul",
    "email":"rahul@example.com"
  }
]
POST

/users

Create a new user.

Request Body

{
  "name":"John",
  "email":"john@example.com"
}

Response

{
  "message":"User created successfully"
}
PUT

/users/{id}

Update an existing user.

DELETE

/users/{id}

Delete a user.

Status Codes

Code Description
200 Success
201 Created
400 Bad Request
401 Unauthorized
404 Not Found
500 Internal Server Error
